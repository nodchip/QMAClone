QMAClone by nodchip (ノドチップ)

概要

GWT(Google Web Toolkit)を用いた クイズマジックアカデミーシミュレーションWEBゲーム

プレイ

ゲームスタートからプレイできます

特徴

  1. 完全オリジナル問題
  2. 問題投稿機能
  3. チャット
  4. ブラウザでプレイ(Win・Mac・UNIX・Linux等でプレイ可)
  5. 全国のプレイヤーとリアルタイム対戦可能

仕様

  • QMA3全国大会モード互換
  • 最大HUM8人同時対戦
  • COMプレイヤー(プレイに応じて強さが変動)
  • ○×・四択・連想・並べ替え・四文字・タイピング・フラッシュ(キューブ)・エフェクト・一問多答・順番当て・線結び・クリック(画面タッチ)・手書き・早い者勝ち・グループ分けを実装完了
  • クラス分け
  • COM対戦・全体対戦・イベント対戦・テーマモード

ヘルプ

こちらをご覧ください

その他

  • フラッシュの表記は「3～4文字→2文字表示」「5～6文字→3文字表示」「7～8文字→4文字表示」となっています
  • アップデート等の都合により突然ゲームが中断することがあります
  • デザイン等は一切考慮しておりません
  • リッチクライアント化(デスクトップソフト化)の予定はありません

  • ザ・マッチメイカァズ - フリー音源を使用させて頂きました
