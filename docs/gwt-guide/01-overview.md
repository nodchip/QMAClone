# 01. GWTドキュメント全体像

対象: DevGuide

## 要点
- GWT公式Docsは「Coding Basics」「Build User Interfaces」「Security」「Optimize」「Reference」に分かれる。
- QMACloneでは次の優先度が高い。
1. Coding Basics: Compile/Debug, Ajax Communication, JsInterop, Deferred Binding
2. Build UI: UIBinder, Cell Tables, CSS Styling
3. Security: SafeHtml, RPC XSRF
4. Optimize: Code Splitting, ClientBundle
5. Reference: Command Line Tools

## QMACloneでの使い方
- 実装前に本索引を起点に、必要トピックだけを辿る。
- 全ページ通読ではなく、機能開発単位で参照する。
